# 题目C — 有符号4位二进制运算

读取最右边4位开关作为4-bit有符号数（符号-绝对值表示法），进行运算和显示。

## 硬件连接

| 外设 | 通道 | 功能 |
|------|------|------|
| GPIO_0 Ch1 | 8-bit 输入 | 拨码开关 |
| GPIO_1 Ch1 | 8-bit 输出 | 七段数码管段选 (共阳极) |
| GPIO_1 Ch2 | 8-bit 输出 | 七段数码管位选 |
| GPIO_2 Ch1 | 5-bit 输入 | 按键 (bit1=L, bit2=C) |

## 符号-绝对值编码规则

```
bit3 = 符号位 (0=正, 1=负)
bit2-0 = 绝对值 (0~7)

例: 0110 = +6    1010 = -2    1100 = -4
```

## 功能描述

| 显示 | 内容 |
|------|------|
| 左4位 (7-4) | 始终显示4位开关的二进制 (0/1)，bit3→最左位 |
| 右2位 (1-0) | C/L 按键控制（初始熄灭） |

| 按键 | 功能 |
|------|------|
| **C** | 最右边2位显示4-bit有符号数的十进制值 |
| **L** | 切换运算: 奇数按=低3位取反(符号不变), 偶数按=原值 |

### 运算示例

| 开关值 | 符号-绝对值 | C显示 | L(奇)低3位取反 | L(偶)原值 |
|--------|-----------|-------|---------------|----------|
| 0110 | +6 | 6 | 0001 → 1 | 6 |
| 1100 | -4 | -4 | 1011 → -3 | -4 |
| 0101 | +5 | 5 | 0010 → 2 | 5 |
| 1010 | -2 | -2 | 1101 → -5 | -2 |

## 三种实现版本

| 版本 | 文件 | 按键处理 |
|------|------|---------|
| V1 | `impl_v1_polling.c` | 扫描体内轮询 |
| V2 | `impl_v2_interrupt.c` | GPIO_2普通中断 |
| V3 | `impl_v3_fast_interrupt.c` | GPIO_2快速中断 |

## 段码 (共阳极)

| 字符 | 段码 | 说明 |
|------|------|------|
| 0 | `0xC0` | |
| 1 | `0xF9` | |
| - | `0xBF` | 仅g段亮 (负号) |
| 熄灭 | `0xFF` | |

数字2-9使用标准共阳极段码。
