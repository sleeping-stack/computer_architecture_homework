# 操作考试复习

MicroBlaze 软核处理器（AC850 开发板）并行 I/O 接口实验复习资料。每道题提供轮询(v1)、普通中断(v2)、快速中断(v3)三种实现，全部使用 `Xil_In32()`/`Xil_Out32()` 直接操作寄存器。

## 硬件映射

| 外设 | 基址宏 | Ch1 | Ch2 | 用途 |
|------|--------|-----|-----|------|
| GPIO_0 | `XPAR_AXI_GPIO_0_BASEADDR` | 8-bit 输入 | 8-bit 输出 | Ch1=拨码开关, Ch2=LED(1=亮) |
| GPIO_1 | `XPAR_AXI_GPIO_1_BASEADDR` | 8-bit 输出 | 8-bit 输出 | Ch1=数码管段选, Ch2=数码管位选 |
| GPIO_2 | `XPAR_AXI_GPIO_2_BASEADDR` | 5-bit 输入 | — | 独立按键(按下=0, 上拉=1) |
| GPIO_3 | `XPAR_AXI_GPIO_3_BASEADDR` | 4-bit 输入 | 4-bit 输出 | 矩阵键盘行/列 |
| Timer | `XPAR_AXI_TIMER_0_BASEADDR` | 100MHz | INT ID=6 | 定时中断 |
| INTC | `XPAR_XINTC_0_BASEADDR` | 快速中断模式 | — | 中断控制器 |

### 按键位映射

| 按键 | GPIO_2 Ch1 位 | 标识 |
|------|--------------|------|
| BTNU | bit0 | U |
| BTNL | bit1 | L |
| BTNC | bit2 | C |
| BTNR | bit3 | R |
| BTND | bit4 | D |

### 数码管

- **类型**: 共阳极 (0=亮, 1=灭)
- **段选**: GPIO_1 Ch1 — a=bit0, b=bit1, c=bit2, d=bit3, e=bit4, f=bit5, g=bit6, dp=bit7
- **位选**: GPIO_1 Ch2 — bit7=最左位, bit0=最右位

## 编码规范

- 只使用 `Xil_In32()` / `Xil_Out32()` 操作寄存器（禁用 `Xil_In8/Out8/In16/Out16`）
- 禁止使用 Vitis 驱动 API（`XGpio_*`, `XUartLite_*`, `usleep` 等）
- 忙等延时使用 `__asm__("nop")` 内联汇编
- 数码管显示使用动态扫描 + 消隐（防止鬼影）
- Timer ISR 中使用查询中断源方式：读 TCSR → 判断 INT_OCCURED → 处理 → 清除
