# 题目B — 滚动显示序列"3456"

合并原题目5+9：8位数码管以1秒间隔滚动显示数字序列"3456"，带方向控制和暂停。

## 硬件连接

| 外设 | 通道 | 功能 |
|------|------|------|
| GPIO_1 Ch1 | 8-bit 输出 | 七段数码管段选 (共阳极, 0=亮) |
| GPIO_1 Ch2 | 8-bit 输出 | 七段数码管位选 (bit7=最左) |
| GPIO_2 Ch1 | 5-bit 输入 | 按键 (bit1=L, bit2=C, bit3=R) |
| Timer | — | 1s 定时周期 |

## 功能描述

| 按键 | 功能 |
|------|------|
| — | 初始: 全部熄灭，不滚动 |
| **C** | 开始/暂停滚动 |
| **L** | 切换为向左滚动 |
| **R** | 切换为向右滚动（默认） |

滚动周期: 1秒/帧，序列"3456"（4位）在8位数码管上移位，8帧一个完整循环。

### 向右滚动示例（默认）

```
F0: [3][4][5][6][_][_][_][_]
F1: [_][3][4][5][6][_][_][_]
...
F4: [_][_][_][_][3][4][5][6]
F5: [6][_][_][_][_][3][4][5]
...
F7: [4][5][6][_][_][_][_][3]
F8: 回到 F0
```

向左滚动方向相反。

## 三种实现版本

| 版本 | 文件 | 定时 | 按键 |
|------|------|------|------|
| V1 | `impl_v1_polling.c` | 忙等计数器 1s | 扫描体内轮询 |
| V2 | `impl_v2_interrupt.c` | Timer普通中断 1s | GPIO_2普通中断 |
| V3 | `impl_v3_fast_interrupt.c` | Timer快速中断 1s | GPIO_2快速中断 |

### 段码 (共阳极)

| 字符 | 段码 |
|------|------|
| 3 | `0xB0` |
| 4 | `0x99` |
| 5 | `0x92` |
| 6 | `0x82` |
| 熄灭 | `0xFF` |
