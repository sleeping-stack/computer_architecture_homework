# 实验任务3 — 矩阵键盘扫描 + 七段数码管 FIFO 显示

## 硬件连接

| 外设 | 通道 | 方向 | 功能 |
|------|------|------|------|
| GPIO_1 Ch1 | 8-bit | 输出 | 七段数码管段选 (a=bit0..dp=bit7, 共阳极 0=亮) |
| GPIO_1 Ch2 | 8-bit | 输出 | 七段数码管位选 (bit7=最左, bit0=最右) |
| GPIO_3 Ch1 | 4-bit (bit3-0) | 输入 | 矩阵键盘行输入 (按下=0, 上拉=1) |
| GPIO_3 Ch2 | 4-bit (bit3-0) | 输出 | 矩阵键盘列输出 (0=选中, 1=不选) |

### 矩阵键盘键位布局

| | Col0 | Col1 | Col2 | Col3 |
|---|---|---|---|---|
| **Row0** | 1 | 2 | 3 | 4 |
| **Row1** | 5 | 6 | 7 | 8 |
| **Row2** | 9 | 0 | A | B |
| **Row3** | C | D | E | F |

### 键盘扫描原理

1. 逐列输出低电平 (0=选中当前列)，其余列输出高电平
2. 读取行输入状态
3. 若某行为低电平，则行列交叉点的按键被按下
4. 查表 `KEY_MAP[row][col]` 得到键值

## 功能描述

1. 扫描 4×4 矩阵键盘，识别 16 个按键 (0~9, A~F)
2. 按键按下的先后顺序显示在 8 位数码管上
3. 先按下的键显示在左侧，新按下的键总是显示在最右侧
4. 当按键超过 8 个时，最早按下的键被左移挤出显示

示例: 依次按下 1→2→3，数码管显示 `[1][2][3][ ][ ][ ][ ][ ]`

## 段码表 (共阳极, 0=亮)

| 字符 | 段码 | 字符 | 段码 |
|------|------|------|------|
| 0 | `0xC0` | 8 | `0x80` |
| 1 | `0xF9` | 9 | `0x90` |
| 2 | `0xA4` | A | `0x88` |
| 3 | `0xB0` | B | `0x83` |
| 4 | `0x99` | C | `0xC6` |
| 5 | `0x92` | D | `0xA1` |

| 熄灭 | `0xFF` | | |

## 三种实现版本

| 版本 | 文件 | 控制方式 | 关键技术 |
|------|------|----------|----------|
| V1 | `impl_v1_polling.c` | 主循环=数码管扫描 + 键盘扫描 | 轮询扫描，边缘检测去抖 |
| V2 | `impl_v2_interrupt.c` | 主循环扫描数码管, Timer 中断触发键盘扫描 | `interrupt_handler`，Timer 50ms 周期 |
| V3 | `impl_v3_fast_interrupt.c` | 主循环扫描数码管, Timer 快速中断触发键盘扫描 | `fast_interrupt`，IMR+IVAR |

### V1 — 轮询

- 主循环以 8 位数码管动态扫描为主体 (~16ms/轮)
- 每 4 轮显示后执行一次完整键盘扫描 (~64ms 间隔)
- 边缘检测去抖: 仅在新按键首次检测到时注册，同一按键持续按下不重复注册
- 键盘扫描: 逐列驱动低电平 → 读行 → 查表

### V2 — 普通中断

- Timer 以 ~50ms (20Hz) 周期产生中断，ISR 中设置键盘扫描标志
- 主循环检测标志后执行键盘扫描和处理
- 主循环持续动态扫描数码管
- ISR 清除 Timer CSR + INTC IAR
- Timer 配置: TLR=4,999,998, 递减模式, 自动重载, 使能中断

### V3 — 快速中断

- 同 V2 逻辑，但 Timer 使用快速中断机制
- 初始化配置 IMR + IVAR
- ISR 仅清除 Timer CSR (INTC 硬件自动应答)
- ISR 声明: `__attribute__((fast_interrupt))`

## 寄存器访问

全部使用 32-bit `Xil_In32()` / `Xil_Out32()` 直接操作寄存器。定时器使用 `xtmrctr_l.h` 中的寄存器偏移和掩码宏。
