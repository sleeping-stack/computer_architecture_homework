# 实验任务2 — 开关输入 + 按键触发 + 七段数码管显示

## 硬件连接

| 外设 | 通道 | 方向 | 功能 |
|------|------|------|------|
| GPIO_0 Ch1 | 8-bit | 输入 | 拨码开关 (SW) |
| GPIO_1 Ch1 | 8-bit | 输出 | 七段数码管段选 (a=bit0..dp=bit7, 共阳极 0=亮) |
| GPIO_1 Ch2 | 8-bit | 输出 | 七段数码管位选 (bit7=最左, bit0=最右) |
| GPIO_2 Ch1 | 5-bit | 输入 | 独立按键 (按下=0, 上拉=1) |

### 按键位映射

| 按键 | GPIO_2 Ch1 位 | 功能 |
|------|--------------|------|
| BTNU | bit0 | 十六进制显示 (低 2 位) |
| BTNC | bit2 | 二进制显示 (全部 8 位) |
| BTND | bit4 | 十进制显示 (低 3 位) |

## 功能描述

1. **BTNC**: 读入 8 位开关值，将每个 bit 的 0/1 显示到全部 8 位数码管上 (bit7→最左位, bit0→最右位)
2. **BTNU**: 读入 8 位开关值，将十六进制数 (00~FF) 显示到低 2 位数码管上，高 6 位熄灭
3. **BTND**: 读入 8 位开关值，将无符号十进制数 (0~255) 显示到低 3 位数码管上，高 5 位熄灭

未按下按键时数码管保持上次显示内容不变。

## 段码表 (共阳极, 0=亮, 1=灭)

| 字符 | 段码 | 字符 | 段码 |

## 三种实现版本

| 版本 | 文件 | 控制方式 | 关键技术 |
|------|------|----------|----------|
| V1 | `impl_v1_polling.c` | 主循环=数码管扫描 + 延时内轮询按键 | 忙等延时，扫描体内读按键/开关 |
| V2 | `impl_v2_interrupt.c` | 主循环扫描数码管，GPIO_2 中断处理按键 | `interrupt_handler`，两级中断清除 |
| V3 | `impl_v3_fast_interrupt.c` | 主循环扫描数码管，GPIO_2 快速中断处理按键 | `fast_interrupt`，IMR+IVAR，自动应答 |

### V1 — 轮询

- 主循环以 8 位数码管动态扫描为主体
- 每位数码管点亮期间 (~2ms)，在忙等循环内轮询按键和开关
- 检测到按键按下后: 读开关 → 更新模式 → 计算段码缓冲 → 等待释放去抖

### V2 — 普通中断

- GPIO_2 中断触发时: ISR 读按键 → 读开关 → 设置模式和更新标志
- 主循环检测更新标志后重新计算段码缓冲
- 主循环持续动态扫描数码管 (延时 + 消隐)
- ISR 清除 GPIO ISR + INTC IAR

### V3 — 快速中断

- 同 V2 逻辑，但使用快速中断机制
- 初始化配置 IMR + IVAR
- ISR 仅清除 GPIO ISR (INTC 硬件自动应答)

## 寄存器访问

全部使用 32-bit `Xil_In32()` / `Xil_Out32()` 直接操作寄存器。
