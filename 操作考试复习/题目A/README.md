# 题目A — Hex显示 + 自动递增 + 手动/自动移位

合并原题目2+3+7：基于8位拨码开关和5个按键的Hex显示综合实验。

## 硬件连接

| 外设 | 通道 | 方向 | 功能 |
|------|------|------|------|
| GPIO_0 Ch1 | 8-bit | 输入 | 拨码开关 |
| GPIO_1 Ch1 | 8-bit | 输出 | 七段数码管段选 (共阳极, 0=亮) |
| GPIO_1 Ch2 | 8-bit | 输出 | 七段数码管位选 (bit7=最左) |
| GPIO_2 Ch1 | 5-bit | 输入 | 独立按键 (按下=0) |
| Timer | — | — | AXI Timer, 100MHz |

按键: bit0=U, bit1=L, bit2=C, bit3=R, bit4=D

## 功能描述

| 按键 | 功能 |
|------|------|
| **C** | 读8位开关值 → 转为2位十六进制(00~FF) → 显示在最低2位数码管（高6位熄灭） |
| **L** | 启动/停止每秒+1自动递增模式。从当前值开始，0→255→0循环。再按停止 |
| **R** | 循环切换自动移位: 关 → 左移(1Hz) → 右移(1Hz) → 关 |
| **U** | 手动左移2位Hex数字（整体），到头循环 |
| **D** | 手动右移2位Hex数字（整体），到头循环 |

Hex显示位置: 2位十六进制数作为一个整体在8位数码管上可移动。初始在最低2位。

## 三种实现版本

| 版本 | 文件 | 定时 | 按键 |
|------|------|------|------|
| V1 | `impl_v1_polling.c` | 忙等计数器模拟1s | 扫描体内轮询 |
| V2 | `impl_v2_interrupt.c` | Timer普通中断(1s) | GPIO_2普通中断 |
| V3 | `impl_v3_fast_interrupt.c` | Timer快速中断(1s) | GPIO_2快速中断 |

### V1 — 轮询
- 主循环=数码管扫描 + 忙等计数模拟1s定时
- 按键在每位扫描延时内轮询，检测后等待释放去抖

### V2 — 普通中断
- Timer ISR (`interrupt_handler`): 1s周期，处理递增和自动移位
- Button ISR (`interrupt_handler`): 处理C/L/R/U/D按键
- 两个ISR均清除外设中断 + INTC IAR

### V3 — 快速中断
- ISR 声明 `fast_interrupt`
- 初始化配置 IMR + IVAR（Timer和GPIO_2各一个）
- ISR 仅清除外设中断（INTC 硬件自动应答）

## Timer 配置
- 时钟: 100MHz
- 1s 周期: TLR = 99,999,998 (= 100M/1 - 2)
- 模式: 递减 + 自动重载 + 使能中断
