# 题目D — LED移位显示

LED读取开关值后进行循环右移操作。

## 硬件连接

| 外设 | 通道 | 功能 |
|------|------|------|
| GPIO_0 Ch1 | 8-bit 输入 | 拨码开关 |
| GPIO_0 Ch2 | 8-bit 输出 | LED (1=亮) |
| GPIO_2 Ch1 | 5-bit 输入 | 按键 (bit2=C, bit3=R) |

## 功能描述

| 操作 | 功能 |
|------|------|
| 初始 | LED 全灭 |
| **C** | 读8位开关值 → 立即显示到全部8位LED |
| **R** | LED整体循环右移一位: 最低位(bit0)移到最高位(bit7) |
| 再次按C | 重新读取开关值更新LED |

### 移位示例

```
LED = 10000001 → 按R → 11000000 → 按R → 01100000 → ...
```

## 三种实现版本

| 版本 | 文件 | 按键处理 |
|------|------|---------|
| V1 | `impl_v1_polling.c` | 主循环轮询 + 忙等去抖 |
| V2 | `impl_v2_interrupt.c` | GPIO_2普通中断 |
| V3 | `impl_v3_fast_interrupt.c` | GPIO_2快速中断 |
