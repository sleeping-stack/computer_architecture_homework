# 实验任务-并行IO接口目录

## [实验任务0]

嵌入式计算机系统将独立按键以及独立开关作为输入设备，LED 灯、七段数码管作为输出设备。LED 灯实时显示独立开光对应位状态，同时8 个七段数码管实时显示最近按下的独立按键位置编码字符（C,U,L,D,R）。

## [实验任务1]

嵌入式计算机系统将独立按键以及独立开关作为输入设备，LED 灯作为输出设备。修改实验示例程序代码，实现以下功能：

1. 点击BTNC 按键时，计算机读入一组8 位独立开关状态作为第一个输入的二进制数据，并即时显示输入的二进制数到8 位LED 灯上。（没有按下BTNC按键时，开关拨动不读入数据）

2. 点击BTNR 按键时，计算机读入另一组8 位独立开关状态作为第二个输入的二进制数据，并即时显示输入的二进制数到8 位LED 灯上。（没有按下BTNR按键时，开关拨动不读入数据）

3. 点击BTNU 按键时，将保存的2 组二进制数据做无符号加法运算，并将运算结果输出到LED 灯对应位。

4. 点击BTND 按键时，将保存的2 组二进制数据做无符号乘法运算，并将运算结果输出到LED 灯对应位。

程序控制方式提示：循环读取按键键值，根据按键的值读取开关状态，并做相应处理。

## [实验任务2]

嵌入式计算机系统将独立按键以及独立开关作为输入设备，七段数码管作为输出设备。实现以下功能：

1. 点击BTNC 按键时，计算机读入一组8 位独立开关状态作为一个二进制数据，并将该二进制数的对应的二进制数值0 或1 显示到8 个七段数码管上。

2. 点击BTNU 按键时，计算机读入一组8 位独立开关状态作为一个二进制数据，并将该16 进制数据各位数字对应的字符0~F 显示到低2 位七段数码管上（高6 位七段数码管不显示）。

3. 点击BTND 按键时，计算机读入一组8 位独立开关状态作为一个二进制数据，并将该数据表示的无符号十进制数各位数字对应的字符0~9 显示到低3 位七段数码管上（高5 位七段数码管不显示）。

程序控制方式提示：程序以七段数码管动态显示控制循环为主体，在循环体内的延时函数内循环读取按键键值以及开关状态，并根据按键值做相应处理。

1. 读取并识别4*4矩阵键盘的各个按键键值（0~F),

2. 按照按键按下的先后顺序逐个将按键的键值显示在8个7段数码管上，要求先按下的显示在左边面，最后按下的显示在右边，每按一个按键，显示键值往左移动一位，即最后按下的键值总是显示在最右侧。

## [题目A] Hex显示 + 自动递增 + 手动/自动移位

嵌入式计算机系统将独立按键以及独立开关作为输入设备，七段数码管作为输出设备。实现以下功能：

1. 点击BTNC按键时，读入8位开关值，转为2位十六进制(00~FF)，显示在最低2位数码管上（高6位熄灭）。

2. 点击BTNL按键时，启动/停止"每秒+1"自动递增模式。从当前开关值开始，每秒+1，到255回0，循环递增。再次按BTNL停止递增。

3. 点击BTNR按键时，循环切换自动移位模式：关 → 左移(1Hz) → 右移(1Hz) → 关。自动移位时，2位Hex数字整体按1Hz频率移动，到边界时循环。

4. 点击BTNU按键时，手动整体左移2位Hex数字，到最左端后回到最右端。

5. 点击BTND按键时，手动整体右移2位Hex数字，到最右端后回到最左端。

程序控制方式提示：使用定时器产生1s周期，结合GPIO中断处理按键。

## [题目B] 滚动显示序列"3456"

嵌入式计算机系统将独立按键作为输入设备，七段数码管作为输出设备。实现以下功能：

1. 初始状态：数码管全部熄灭，不滚动。

2. 点击BTNC按键时，开始/暂停滚动。

3. 点击BTNL按键时，切换为向左滚动。

4. 点击BTNR按键时，切换为向右滚动（默认方向）。

5. 滚动方式：数字序列"3→4→5→6"（4位）在8位数码管上整体移动，每秒移动一位，8帧一个完整循环。没有数字的位置熄灭。

程序控制方式提示：使用定时器产生1s滚动周期，每次移位1位。

## [题目C] 有符号4位二进制运算

嵌入式计算机系统将独立按键以及独立开关作为输入设备，七段数码管作为输出设备。实现以下功能：

1. 读取最右边4位开关作为4-bit有符号数（符号-绝对值表示法：bit3=符号位(0=正,1=负)，bit2-0=绝对值(0~7)）。最左边4位数码管始终显示这4位的二进制值(0/1)。

2. 点击BTNC按键时，最右边2位数码管显示该4-bit有符号数的十进制值。例：0110→"6"，1010→"-2"，1100→"-4"。

3. 点击BTNL按键时，在两种运算间切换（基于原始开关值）：
   - 奇数按：符号位不变，低3位取反，显示符号-绝对值十进制值。例：0110→0001="1"，1100→1011="-3"。
   - 偶数按：显示原始值的符号-绝对值十进制（与C显示相同）。

程序控制方式提示：循环读取按键和开关值，根据按键更新显示模式。

## [题目D] LED移位显示

嵌入式计算机系统将独立按键以及独立开关作为输入设备，LED灯作为输出设备。实现以下功能：

1. 初始状态：全部LED灯熄灭。

2. 点击BTNC按键时，读8位开关值，显示到全部8位LED（按下瞬间捕获）。

3. 点击BTNR按键时，LED整体循环右移一位。最低位(bit0)移到最高位(bit7)。再次按C可重新读取开关值。

例：LED=10000001 → 按R → 11000000 → 按R → 01100000 → ...

程序控制方式提示：循环读取按键键值，根据按键值做相应处理。

## [题目E] 综合多功能题

嵌入式计算机系统将独立按键以及独立开关作为输入设备，LED灯、七段数码管作为输出设备。5个按键分别控制5种独立功能：

**A. BTNC — 流水灯速度切换**
数码管熄灭。LED单灯右移流水：1s → 0.5s → 0.25s → 1s循环切换速度。单灯从高位向低位循环移动。

**B. BTNL — LED移位**
数码管熄灭。第1次按BTNL：LED显示8位开关当前状态。之后每次按BTNL：LED整体左移一位（循环），到头从右边进入。

**C. BTND — 开关二进制+LED奇偶**
左边4位数码管显示最右边4位开关的二进制值(0/1)。右边4位熄灭。LED奇数位常亮(bits 1,3,5,7=1)，偶数位熄灭。

**D. BTNR — 开关反码+LED奇偶反转**
右边4位数码管显示最右边4位开关的逐位反码。左边4位熄灭。LED偶数位常亮(bits 0,2,4,6=1)，奇数位熄灭。

**E. BTNU — 有符号十进制滚动**
最右边4位开关的有符号十进制值（符号-绝对值，-7~+7）在8位数码管上滚动显示。按BTNU切换滚动速度：1s/0.5s。LED实时显示8位开关状态。

例：开关为1011→"-3"在数码管滚动；0110→"6"滚动；1100→"-4"滚动。

程序控制方式提示：使用定时器产生0.25s基准周期，ISR中分频实现多速率定时(1s/0.5s/0.25s)。GPIO中断处理按键切换模式。
