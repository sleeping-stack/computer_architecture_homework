# 题目E — 综合多功能题

5个按键分别控制5种独立功能，综合考察LED、数码管、按键、定时器的联合使用。

## 硬件连接

| 外设 | 通道 | 功能 |
|------|------|------|
| GPIO_0 Ch1 | 8-bit 输入 | 拨码开关 |
| GPIO_0 Ch2 | 8-bit 输出 | LED (1=亮) |
| GPIO_1 Ch1 | 8-bit 输出 | 七段数码管段选 (共阳极, 0=亮) |
| GPIO_1 Ch2 | 8-bit 输出 | 七段数码管位选 (bit7=最左) |
| GPIO_2 Ch1 | 5-bit 输入 | 按键 |
| Timer | — | 0.25s 基准定时 |

## 功能描述

| 按键 | 模式 | 功能 |
|------|------|------|
| **C** | A.流水灯 | 数码管熄灭。LED单灯右移流水: 1s→0.5s→0.25s→1s速度循环切换 |
| **L** | B.LED移位 | 数码管熄灭。第1次按=LED显示开关值。之后每次=LED整体左移1位(循环) |
| **D** | C.二元显示 | 左4位=开关低4位二进制(0/1)。右4位=熄灭。LED: 奇数位常亮(bits 1,3,5,7=1) |
| **R** | D.反码显示 | 右4位=开关低4位逐位反码。左4位=熄灭。LED: 偶数位常亮(bits 0,2,4,6=1) |
| **U** | E.滚动 | 开关低4位有符号十进制值(-7~+7)在8位数码管滚动。LED=开关全值。1s/0.5s切换 |

### U按键滚动说明

开关低4位按符号-绝对值编码(bit3=符号, bit2-0=绝对值):
- 1011 → 显示 "-3" (3个字符)
- 0110 → 显示 "6" (1个字符)
- 1100 → 显示 "-4"
- 在8位数码管上整体滚动，到头循环

## 三种实现版本

| 版本 | 文件 | 定时 | 按键 |
|------|------|------|------|
| V1 | `impl_v1_polling.c` | 忙等计数器 (~16ms/tick) | 扫描体内轮询 |
| V2 | `impl_v2_interrupt.c` | Timer普通中断(0.25s) | GPIO_2普通中断 |
| V3 | `impl_v3_fast_interrupt.c` | Timer快速中断(0.25s) | GPIO_2快速中断 |

### Timer多速率分频

| 动作 | 周期 | 分频 |
|------|------|------|
| 0.25s | 1 tick | 每次ISR |
| 0.5s | 2 ticks | tick % 2 == 0 |
| 1s | 4 ticks | tick % 4 == 0 |
